# Cogna Docs

这是 Cogna 的文档站工程。

## 本地运行

```bash
pnpm dev
```

## 构建

```bash
pnpm build
```

## 文档分工

- 用户文档：`src/content/docs/`
- 贡献者文档：`src/content/contrib/`
- 仓库级贡献说明：`../CONTRIBUTING.md`

## 当前信息架构

- 顶部导航：快速开始 / 用户文档 / 贡献者文档 / 开发进度 / GitHub
- 用户文档采用 Divio 四象限：Tutorials / How-to / Reference / Explanation
- 主页 feature 区用于展示四象限和当前核心能力
