# CodeIQ Docs

这是 CodeIQ 的文档站工程。

## 本地运行

```bash
pnpm dev
```

## 构建

```bash
pnpm build
```

## 主要内容位置

- 页面内容：`src/content/docs/`
- 静态资源：`public/`

## 当前文档主题

文档内容围绕以下主题组织：

- CodeIQ 的产品定位与适用场景
- `CIQ Bundle`、`CIQ Diff`、`SARIF` 等核心产物
- CLI 工作流：`init / build / diff / check / query / publish`
- MCP 查询工作流：`codeiq.query.outlines` → `codeiq.query.symbol`
- 功能成熟度与路线图
