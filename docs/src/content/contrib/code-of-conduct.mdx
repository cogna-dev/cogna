# 行为准则

本页是贡献者附录落点，用来说明当前仓库工作树内的文档状态。

## 当前状态

当前仓库根目录下 #strong[没有单独的 `CODE_OF_CONDUCT.md` 文件] 供本页直接引用。

## 当前最小协作预期

在缺少独立行为准则文档的情况下，贡献者至少应遵守：

- 在 issue、PR 和讨论中保持专业与可协作；
- 讨论实现和文档时聚焦事实、证据与可复现结果；
- 避免把内部争论直接泄漏到用户文档；
- 对用户与其他贡献者保持尊重。
