# CLA

本页是贡献者附录落点，用来说明当前仓库工作树内的文档状态。

## 当前状态

当前仓库根目录下 #strong[没有单独的 CLA 文件] 供本页直接引用。

这意味着：当前文档站只能把它作为附录落点，而不能把某份 CLA 条款原文展示在这里。

## 对贡献者意味着什么？

如果你的组织需要在合并前确认 CLA / DCO / 授权条款，请在提交重要变更前先和维护者确认，而不要假设仓库已经自动配置了独立 CLA 流程。
