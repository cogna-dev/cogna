# 开源协议

本页是贡献者附录落点，用来说明当前仓库工作树内的文档状态。

## 当前状态

当前仓库根目录下 #strong[没有单独的 `LICENSE` 文件] 供本页直接引用。

因此，这一页只说明现状，而不替代正式 license text。

## 对贡献者意味着什么？

如果你要：

- 复制仓库代码到别处；
- 重新分发打包结果；
- 为外部发布做合规确认；

请先确认项目在代码托管平台上的仓库元数据或与维护者核实，而不要只依赖本页。
