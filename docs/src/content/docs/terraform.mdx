# Terraform

Terraform module 的升级风险通常来自输入输出契约变化。Cogna 可以帮助你提前发现。

## 推荐配置

```yaml
profile: terraform-module
inputs:
  include:
    - "**/*.tf"
```

## 常见高风险变化

- variable 从可选改为必填
- output 被删除或类型变化
- provider 约束收紧

## 使用步骤

```bash
cogna build
cogna diff
cogna check
```
