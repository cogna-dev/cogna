# 开发进度

这页只回答一个问题：你现在能用 Cogna 做什么。

## 已可用能力

- 安装方式：Desktop、CLI、SDK、VSCode（见 [/download](/download)）
- 基础流程：`init`、`build`、`diff`、`check`
- 查询能力：MCP 与签名检索
- 团队协作：HTTP cache 复用
- 质量集成：SARIF 输出与 CI 门禁

## 推荐采用顺序

1. 个人先跑通 [快速开始](/docs/quickstart)
2. 团队统一 [配置与 PURL](/docs/config)
3. 把检查接入 [持续集成](/docs/ci)
4. 再按需要接入 [SDK 使用指南](/docs/sdk) 与 [AI 查询（MCP）](/docs/mcp)

## 适配语言

- Go
- Rust
- Terraform
- OpenAPI

## 适合谁

- 维护公共 API 的团队
- 需要发布前兼容性门禁的项目
- 希望用统一事实源给 AI/IDE/CI 提供上下文的团队

## 下一步

- 如果你还没安装：先去 [/download](/download)
- 如果你已安装：直接从 [快速开始](/docs/quickstart) 开始
