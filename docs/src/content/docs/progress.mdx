## 已完成阶段（摘要）

- `R1-R4` 已完成并冻结为当前 repo 基线；
- `R5` 已完成，主业务流程 `init → build → diff → check → query → MCP review` 已形成稳定闭环；
- 当前看板不再保留 `R5` 的执行清单，只保留当前 active release（`R6`）的唯一执行清单。

## R6 · Single-Machine Registry Baseline（Active）

**Scope**

- 收口单机、免认证 registry baseline，聚焦本地 artifact movement：
  - `publish` 写入本地 registry；
  - `registry upload-local / download-local / download(materialize)` 可回归；
  - upload/download/receipt contract 与命令行为保持一致。

**Done when**

- `publish ↔ registry(upload-local/download-local/download)` 单机闭环稳定；
- `codeiq-registry-upload/v1`、`codeiq-registry-download/v1`、`codeiq-publish-receipt/v1` contract 与实现一致；
- R6 对外文档（plan/progress/CLI）和验证命令保持一致。

**Verify**

- `moon test src/cmd/registry/main_test.mbt --target wasm -v`
- `moon test src/cmd/registry/download_bundle_test.mbt --target wasm -v`
- `moon test src/cmd/publish/main_test.mbt --target wasm -v`
- `moon test src/e2e/registry_test.mbt --target wasm -v`
- `moon test src/schema/contracts_test.mbt --target native -v`
- `pnpm build`（在 `docs/` 下）

**Out of scope**

- registry HTTP server productization、shared verifier、hosted deployment、KMS/key rotation。
