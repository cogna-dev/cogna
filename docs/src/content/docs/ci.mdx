# 持续集成

把 Cogna 接入 CI 后，你可以在每个 PR 上自动获得兼容性反馈。

## 最小流水线

```yaml
- uses: ./integrations/actions/setup-cogna
- uses: ./integrations/actions/run-cogna
  with:
    working-directory: ./sdk
    command: build
- uses: ./integrations/actions/run-cogna
  with:
    working-directory: ./sdk
    command: diff
- uses: ./integrations/actions/run-cogna
  with:
    working-directory: ./sdk
    command: check
    upload-sarif: true
```

## 推荐产物

- `dist/bundle.ciq.tgz`
- `dist/diff.json`
- `dist/check.sarif.json`

## 分阶段落地建议

1. 第一阶段：只上传结果，不阻断合并
2. 第二阶段：对 `error` 启用阻断
3. 第三阶段：形成发布门禁基线
