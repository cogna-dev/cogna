# SKILL 集成

SKILL 可以理解为“任务模板”。它把常见操作（例如构建、对比、检查、查询）固化成一套可重复流程，减少遗漏。

## 适用场景

- 让 AI 助手按统一步骤执行任务
- 减少不同成员之间的操作差异
- 把经验流程沉淀成可复用规范

## 推荐流程

1. 安装并配置 Cogna（见 [/download](/download) 与 [快速开始](/docs/quickstart)）
2. 选择对应任务模板（构建、diff、check、查询）
3. 固定输入输出路径（如 `dist/`）
4. 在 CI 中复用同一流程

## 成功标准

- 相同输入得到稳定输出
- 结果能被团队复核
- 失败时可以快速定位步骤
