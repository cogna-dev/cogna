# 支持范围

这一页属于 #strong[Explanation]：它解释当前适用边界，而不是逐条列参数。

## 当前支持

- Go module
- Rust crate
- Terraform Module
- OpenAPI specification
- OPA policy bundle

## 各 profile 当前关注什么？

| Profile | 当前关注的公开事实 |
| --- | --- |
| `go-module` | 导出函数、类型、常量、接口、方法集等声明变化 |
| `rust-crate` | public item、trait / impl、unsafe / extern、where clause 等声明变化 |
| `terraform-module` | variable、output、module、provider、lifecycle 与 meta-argument 契约 |
| `openapi-spec` | path、operation、parameter、request / response、schema 等 API contract |
| `policy-bundle` | Rego policy、OPA bundle entrypoint、policy catalog |

## 当前适合的使用方式

- 本机 build / diff / check / query
- 本机共享 bundle
- 本机 AI / Agent 查询

## 当前支持到什么程度？

- 代码 / 规格 repo 可以稳定 build 成 `CIQ Bundle`
- `diff` 已经能为多种生态输出 richer semantic categories
- `check` 已经能把 diff + policy 结果转成最小 SARIF
- built-in compatibility policy baseline 正在继续扩展，但当前用户流程仍然是显式传入本地 policy bundle

## 当前不在范围内

- hosted registry
- distributed replication
- multi-node HA
- trust verifier / KMS

## 什么时候看这页？

- 你在判断项目能不能接入 CodeIQ
- 你在确认当前边界
- 你需要区分用户能力和未来方向
