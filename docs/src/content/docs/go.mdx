# Go

Go 是 Cogna 的常见使用场景之一，适合做库版本升级前的兼容性把关。

## 推荐配置

```yaml
profile: go-module
inputs:
  include:
    - "**/*.go"
```

## 使用步骤

```bash
cogna build
cogna diff
cogna check
```

## 重点关注变化

- 导出函数签名变化
- 导出结构体字段变化
- 接口方法集变化
- 删除公开符号
