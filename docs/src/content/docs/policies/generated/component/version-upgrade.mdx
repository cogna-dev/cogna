# compat.component.version-upgrade

## 摘要

Software component version changed.

## 元数据

| 字段 | 值 |
|------|----|
| `ruleId` | `compat.component.version-upgrade` |
| `family` | `component` |
| `defaultLevel` | `warning` |
| `entrypoint` | `data.cogna.compat.deny` |
| `helpUri` | `https://cogna.xaclabs.dev/docs/policies/generated/component/version-upgrade` |

## 触发条件

当 OPA compatibility policy 命中该规则的谓词时，会在 `data.cogna.compat.deny` 下返回结果。

## SARIF 示例

```json
{
  "ruleId": "compat.component.version-upgrade",
  "level": "warning",
  "message": {
    "text": "Software component version changed."
  },

}
```
