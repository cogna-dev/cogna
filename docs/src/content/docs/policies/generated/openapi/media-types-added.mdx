# compat.openapi.media-types-added

## 摘要

OpenAPI media types were added.

## 元数据

| 字段 | 值 |
|------|----|
| `ruleId` | `compat.openapi.media-types-added` |
| `family` | `openapi` |
| `defaultLevel` | `note` |
| `entrypoint` | `data.cogna.compat.deny` |
| `helpUri` | `https://cogna.xaclabs.dev/docs/policies/generated/openapi/media-types-added` |

## 触发条件

当 OPA compatibility policy 命中该规则的谓词时，会在 `data.cogna.compat.deny` 下返回结果。
