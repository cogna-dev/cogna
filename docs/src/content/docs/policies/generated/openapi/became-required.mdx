# compat.openapi.became-required

## 摘要

OpenAPI field or parameter became required.

## 元数据

| 字段 | 值 |
|------|----|
| `ruleId` | `compat.openapi.became-required` |
| `family` | `openapi` |
| `defaultLevel` | `error` |
| `entrypoint` | `data.codeiq.compat.deny` |
| `helpUri` | `https://codeiq.xaclabs.dev/docs/policies/generated/openapi/became-required` |

## 触发条件

当 `semanticDiff.findings[]` 中出现 `openapi-became-required` 时，这条规则会在 `data.codeiq.compat.deny` 下产生命中。
