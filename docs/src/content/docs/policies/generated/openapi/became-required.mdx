# compat.openapi.became-required

## 摘要

OpenAPI field or parameter became required.

## 元数据

| 字段 | 值 |
|------|----|
| `ruleId` | `compat.openapi.became-required` |
| `family` | `openapi` |
| `defaultLevel` | `error` |
| `entrypoint` | `data.cogna.compat.deny` |
| `helpUri` | `https://cogna.xaclabs.dev/docs/policies/generated/openapi/became-required` |

## 触发条件

当 OPA compatibility policy 命中该规则的谓词时，会在 `data.cogna.compat.deny` 下返回结果。

## SARIF 示例

```json
{
  "ruleId": "compat.openapi.became-required",
  "level": "error",
  "message": {
    "text": "OpenAPI field or parameter became required."
  },
  "helpUri": "https://cogna.xaclabs.dev/docs/policies/generated/openapi/became-required"
}
```
