# compat.openapi.response-schema-narrowed

## 摘要

OpenAPI response schema narrowed and clients may fail to parse it.

## 元数据

| 字段 | 值 |
|------|----|
| `ruleId` | `compat.openapi.response-schema-narrowed` |
| `family` | `openapi` |
| `defaultLevel` | `error` |
| `entrypoint` | `data.cogna.compat.deny` |
| `helpUri` | `https://cogna.xaclabs.dev/docs/policies/generated/openapi/response-schema-narrowed` |

## 触发条件

当 OPA compatibility policy 命中该规则的谓词时，会在 `data.cogna.compat.deny` 下返回结果。
