# compat.terraform.input-became-required

## 摘要

Terraform input became required and callers must update configuration.

## 元数据

| 字段 | 值 |
|------|----|
| `ruleId` | `compat.terraform.input-became-required` |
| `family` | `terraform` |
| `defaultLevel` | `error` |
| `entrypoint` | `data.cogna.compat.deny` |
| `helpUri` | `https://cogna.xaclabs.dev/docs/policies/generated/terraform/input-became-required` |

## 触发条件

当 OPA compatibility policy 命中该规则的谓词时，会在 `data.cogna.compat.deny` 下返回结果。
