# Rust

如果你维护 Rust crate，Cogna 可以帮你把“代码改动”转换成“对调用方的影响”。

## 推荐配置

```yaml
profile: rust-crate
inputs:
  include:
    - "**/*.rs"
```

## 使用步骤

```bash
cogna build
cogna diff
cogna check
```

## 重点关注变化

- `pub` 类型和函数签名变化
- trait 对外行为变化
- 删除或重命名公开 API
