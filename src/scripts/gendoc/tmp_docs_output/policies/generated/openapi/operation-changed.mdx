# compat.openapi.operation-changed

## 摘要

OpenAPI operation changed.

## 元数据

| 字段 | 值 |
|------|----|
| `ruleId` | `compat.openapi.operation-changed` |
| `family` | `openapi` |
| `defaultLevel` | `warning` |
| `entrypoint` | `data.codeiq.compat.deny` |
| `helpUri` | `https://codeiq.xaclabs.dev/docs/policies/generated/openapi/operation-changed` |

## 触发条件

当 `semanticDiff.findings[]` 中出现 `openapi-operation-changed` 时，这条规则会在 `data.codeiq.compat.deny` 下产生命中。
