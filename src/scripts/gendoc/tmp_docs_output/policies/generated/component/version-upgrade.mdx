# compat.component.version-upgrade

## 摘要

Software component version changed.

## 元数据

| 字段 | 值 |
|------|----|
| `ruleId` | `compat.component.version-upgrade` |
| `family` | `component` |
| `defaultLevel` | `warning` |
| `entrypoint` | `data.codeiq.compat.deny` |
| `helpUri` | `https://codeiq.xaclabs.dev/docs/policies/generated/component/version-upgrade` |

## 触发条件

当组件级 diff 的 `componentDiff.upgradeKind` 等于 `version-upgrade` 时，这条规则会在 `data.codeiq.compat.deny` 下产生命中。
